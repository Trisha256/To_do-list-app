# To-do List App

A simple **Task Manager / To‑Do List** web app built with **HTML, CSS, and JavaScript**.  
It supports saving tasks in your browser using **localStorage**, marking tasks as **completed**, filtering tasks, and **editing** task names.

## Live Demo
If you enabled GitHub Pages, your live site will look like:

- `https://trisha256.github.io/To_do-list-app/`

(If you don’t see it, go to **Repo → Settings → Pages → Deploy from branch → main → /(root)**.)

## Features
- Add new tasks
- Delete tasks
- Tasks persist after refresh using **localStorage**
- Mark tasks as **Completed**
- Filter by:
  - **All**
  - **Active**
  - **Completed**
- Edit task text (Save / Cancel)

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## Project Structure
- `index.html` – app layout
- `style.css` – styling
- `script.js` – functionality (add/delete/save/filter/edit)

## How to Run Locally
1. Download or clone the repository
2. Open `index.html` in your browser

Optional (recommended): run using a simple local server:
- VS Code → install **Live Server** extension → Right‑click `index.html` → **Open with Live Server**

## How It Works (localStorage)
The app stores your tasks in `localStorage`, so tasks remain even if you refresh the page or close the browser.
